# CoPharma.ai

موقع صيدلي ذكي من تصميم إبراهيم، صيدلي سريري في أبوظبي.

## التقنيات المستخدمة
- Next.js
- Tailwind CSS
- نشر على Vercel

## خطوات التشغيل
1. npm install
2. npm run dev
3. نشر على Vercel عبر GitHub

## الصفحات
- الصفحة الرئيسية
- الخدمات
- المنتجات
- من أنا
- الحجز
- المدونة
- التواصل
- سياسة الخصوصية
- تقييمات العملاء

## الرؤية
